# 🌾 Crop Yield Prediction using Machine Learning

## 📌 Overview
This project predicts crop yield using machine learning based on environmental and agricultural factors. It helps in improving decision-making in smart agriculture.

---

## 🎯 Objective
To build a predictive model using features like rainfall, temperature, fertilizer usage, and crop type.

---

## 📊 Dataset Features
- State
- Crop
- Season
- Area
- Rainfall
- Temperature
- Fertilizer
- Pesticide

---

## ⚙️ Approach
1. Data preprocessing (handling categorical features using one-hot encoding)
2. Train-test split
3. Model training using Random Forest
4. Model comparison with Linear Regression
5. Evaluation using RMSE and R² score
6. Visualization and feature importance analysis

---

## 🤖 Models Used
- Random Forest Regressor
- Linear Regression

---

## 📈 Results
- R² Score: ~0.96
- RMSE: ~165

The model shows strong predictive performance.

---

## 🔍 Key Insights
- Rainfall and temperature are major factors affecting yield
- Fertilizer and pesticide usage also play a significant role

---
